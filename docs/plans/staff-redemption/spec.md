# Staff Redemption Specification

## Overview
Store staff validate issued voucher codes or QR tokens and mark valid vouchers as redeemed.

## Features
F-STAFF-01.

## API Mapping
| Feature | API |
|---|---|
| Validate voucher | POST /api/staff/vouchers/validate |
| Redeem voucher | POST /api/staff/vouchers/redeem |
