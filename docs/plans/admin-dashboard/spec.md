# Admin Dashboard Specification

Campaign operators monitor funnel activity, slot performance, benefit performance, and export voucher data.

## Features

## API Mapping

|---|---|
| Metrics | GET /api/dashboard/campaigns/[id] |
| CSV export | GET /api/export/campaigns/[id] |
