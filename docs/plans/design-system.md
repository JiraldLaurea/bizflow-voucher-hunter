# Design System

## Color Tokens
- Navy: `#0B1D3A`
- Purple Primary: `#5C3DFF`
- Purple Secondary: `#7C4DFF`
- Canvas: `#F7F9FD`
- Surface: `#FFFFFF`
- Line: `#E5E9F2`
- Success: `#22C55E`
- Warning: `#F59E0B`
- Danger: `#EF4444`

## Components
- Mobile phone frame: 24px radius, white background, raised shadow.
- Admin cards: 12px radius, subtle border, soft shadow.
- Buttons: purple gradient primary, white outlined secondary.
- Chips: pill badges for active, low stock, sold out, selected, redeemed.

## Typography
- Header H1: bold, 28-58px depending on viewport.
- Header H2: 20-24px.
- Body: 14px.
- Caption: 12px.

## UI Direction
Follow the supplied mockups: clean fintech/SaaS interface, navy and purple control palette, compact cards, mobile-first public flow, dense dashboard tables, and dark admin sidebar.
